2 Copy

copy方法返回一个具有相同键值对的新字典（浅复制），示例代码如下：

mydict.copy()

copy方法返回mydict的地址副本(Shallow copy)，如果替换了地址副本中值的地址，新旧dict间不会产生影响，但如果对地址副本中地址对应的值作直接修改，则会同时影响新旧dict，其影响可以用deep copy的方法来避免

Setdefault

若给定键存在，返回给定键对应的值，若给定键不存在，返回默认值，并根据给定键和默认值更新字典。
